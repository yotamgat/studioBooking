### 3. 💰 מחירונים גמישים

**מה נוסף:**

#### מבנה מחירון (`pricingTiers`):
```typescript
{
  name: 'שם המחירון',          // לדוגמה: "חבילת 2-4 שעות"
  description: 'תיאור',        // לדוגמה: "חיסוך של 10%"
  minHours: 2,                 // מינימום שעות
  maxHours: 4,                 // מקסימום שעות (אופציונלי)
  pricePerHour: 135,           // מחיר לשעה
  discount: 10                 // אחוז הנחה (אופציונלי)
}
```

#### דוגמאות למחירונים שנוצרו:

**חלל 1 (הגדול):**
- שעה בודדת: ₪150/שעה
- 2-4 שעות: ₪135/שעה (חיסוך 10%)
- 5+ שעות: ₪120/שעה (חיסוך 20%)

**חלל 2 (האינטימי):**
- שעה בודדת: ₪120/שעה
- 2-3 שעות: ₪108/שעה (חיסוך 10%)
- 4+ שעות: ₪102/שעה (חיסוך 15%)

#### חישוב מחיר אוטומטי:
- ✅ הסיסטם בוחר אוטומטית את המחירון המתאים
- ✅ מציג את המחיר הסופי בעמוד האישור
- ✅ מראה את ההנחה אם יש

## 🎨 מה תראה עכשיו

### בעמוד בחירת חלל:

1. **כרטיסי חללים גדולים** עם:
   - תמונה ראשית גדולה
   - אינדיקטור "עוד X תמונות"
   - כותרת וסיקור
   - אייקונים: קיבולת (👥) וגודל (📐)
   - תגיות תכונות (features)
   
2. **מחירונים** מוצגים בצורה ברורה:
   - שם המחירון
   - טווח שעות
   - מחיר לשעה
   - אחוז חיסוך (אם יש)

3. **פרטים נוספים** (מתרחב בלחיצה):
   - מידע מפורט על החלל
   - כל המתקנים
   - תנאים

4. **כפתור בחירה**:
   - משתנה צבע כשנבחר
   - הודעה "נבחר ✓"

### בעמוד אישור:

- ✅ חישוב מחיר אוטומטי לפי מחירון
- ✅ הצגת "מחיר מיוחד לפי מחירון חבילות"
- ✅ תצוגה מפורטת של החלל

## 🖼️ ניהול תמונות

### כרגע (פיתוח):
התמונות מגיעות מ-Unsplash (URLs חיצוניים)

### עתידית (פרודקשן):
ראה את `IMAGE_UPLOAD_GUIDE.md` לאפשרויות:
1. Cloudinary (מומלץ)
2. Vercel Blob Storage
3. AWS S3

### להחלפת תמונות זמנית:
1. מצא תמונה ב-[Unsplash](https://unsplash.com/s/photos/dance-studio)
2. לחץ ימני → "Copy Image Address"
3. ערוך `scripts/seed.js`
4. הכנס את ה-URL למערך `images`
5. הרץ `node scripts/seed.js`
